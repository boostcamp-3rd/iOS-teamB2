### 보다비

Boostcamp iOS 희노애락팀 프로젝트

##### 기획 의도

경조사 종이 장부를 다시 찾아보지 않아도 언제 어디서든 감사했던 사람들을 검색하고 알림을 받는다

나의 생일을 챙겨준 고마운 사람들을 기록하고, 그 사람들의 생일에 잊지않고 보답한다



##### 핵심 기능

* 효율적인 기록 방법 제공
* iCloud 백업 기능
* 지인의 경조사 알림 기능
* 기타 기능 추가 예정
